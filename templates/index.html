// объединение отчётов FB + Keitaro

01 — Facebook отчёты
Выгрузка по креативам из Ads Manager. Можно загрузить несколько файлов с разных кабинетов — спенд и клики просуммируются автоматически.
📊
Выберите файлы .xlsx можно несколько файлов сразу
02 — Keitaro отчёт
Отчёт по креативам с регистрациями и депозитами. Выгружается как CSV с разделителем точка с запятой.
📈
Выберите файл .csv один файл
Требования к файлам
Facebook · .xlsx
Колонка 1 Ad name — название креатива
Колонка 2 Amount spent — потраченный бюджет в USD
Колонка 3 Impressions — количество показов
Колонка 4 Unique link clicks — уникальные клики
Keitaro · .csv
Колонка 1 Sub ID — название креатива (должно совпадать с Ad name из FB)
Колонка 3 Conv. — количество регистраций
Колонка 5 Revenue — доход
Колонка 9 Deposits — количество депозитов
⚠ Важно Названия креативов в FB (Ad name) и Keitaro (Sub ID) должны совпадать точно — включая регистр и пробелы. Иначе строки не смержатся и депозиты не подтянутся.
